# MulCryptDB系统介绍文档

## 一、系统概述

本系统是面向语义检索的多模态加密数据系统，基于 SM4 商用密码算法与后量子 SLH-DSA 签名算法构建全密态安全检索体系，实现图像、音频、文本等跨模态数据的密态关联分析与语义检索。系统采用对称可搜索加密与 SLH-DSA 签名双重机制，结合容错码与差分隐私技术，融合 ImageBind 大模型实现跨模态哈希检索，支持十亿级密态数据的百毫秒级语义响应，综合性能损失控制在 10% 以内，满足后量子密码标准抗攻击要求。

## 二、前端文件架构

### 1. HTML 页面组件

- **首页 (index.html)**：系统入口页面，展示多模态加密数据库核心优势，包括防量子攻击密文索引体系、容错编码隐私保护及跨模态哈希检索能力，提供五大检索模型导航入口。
- **管理员控制台 (admin_dashboard.html)**：可视化管理界面，支持音频 / 图像文件的网格 / 表格视图切换、多条件排序、分页浏览及加密 / 原始文件预览，集成文件上传与状态监控功能。
- **管理员控制台登录 (admin_login.html)**：管理员登录界面。
- **五大检索模型页面 (model1-5.html)**：分别实现图像→音频、音频→图像、文本→多模态、音文联合、音图联合检索功能，展示检索过程中的性能指标、加密令牌生成及哈希匹配结果，支持加密资源可视化预览。

### 2. 样式系统 (styles.css)

统一管理系统视觉风格，包含：

- 响应式布局组件（适配移动端与桌面端）
- 拖拽交互样式（文件上传区域视觉反馈）
- 模态框与预览组件样式
- 检索过程日志展示样式
- 管理员控制台功能区布局样式

### 3. 交互逻辑 (JavaScript 文件)

- **模型交互脚本 (model1-5.js)**：处理用户输入（文件上传、文本检索），与后端实时通信获取检索过程流数据，动态更新页面性能指标、加密过程日志及检索结果预览。
- **管理控制台脚本 (admin_dashboard.js)**：实现文件列表动态加载、类型 / 视图切换、排序分页控制，集成登录状态验证与错误处理机制。
- **登录逻辑脚本 (admin_login.js)**：处理管理员身份验证，支持账号密码本地存储与会话状态管理。

## 三、前后端通信机制

后端服务基于 Flask 框架构建，通过 RESTful 接口与前端交互：

- **流式处理**：采用 Server-Sent Events (SSE) 技术，实时返回检索过程日志（特征提取、令牌生成、哈希匹配等阶段状态）。
- **加密服务**：提供 SM4 算法加密的图像 / 音频文件存储与检索接口，自动生成 SLH-DSA 签名保障数据完整性。
- **资源访问**：支持加密 / 原始文件的下载预览，确保密态数据全生命周期安全。

- **浏览器支持**：Chrome 60+、Firefox 55+、Safari 11+、Edge 79+（需支持 ES6 + 特性与 Fetch API）
